# PagerMaid

Pagermaid 是一个用在 Telegram 的实用工具。

它通过响应账号通过其他客户端发出的命令来自动执行一系列任务。

原英文更新频道: https://t.me/PagerMaid

## 安装

请阅读 [INSTALL.md](https://github.com/xtaodada/PagerMaid-Modify/blob/master/INSTALL.md)

## 与原作者联络

您可以在 Telegram 上使用 -contact <message> 并点击链接进入通过 Pagermaid 自动打开的 PM ，如果您安装上出现了问题，请通过 [stykers@stykers.moe](mailto:stykers@stykers.moe) 给我发电子邮件，或在 Telegram [@KatOnKeyboard]（https://t.me/KatOnKeyboard）上给我发消息。
